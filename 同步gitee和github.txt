//git默认远程库名称为origin
git remote rm origin

//关联gitee并设置别名为gitee
git remote add gitee @git/gitee.com:admin/demo.git
//关联github并设置别名为github
git remote add githob @git/github.com:admin/demo.git

推送到Gitee
$ git push gitee master
推送到GitHub
$ git push github master
